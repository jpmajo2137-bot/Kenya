# 케냐어(스와힐리어) 단어 암기 앱

`C:\cursor app\kenya-vocab`에 생성된 **웹 기반 단어 암기 앱**입니다.  
요청하신 “케냐어”는 실사용 관점에서 **스와힐리어(Swahili)** 기본 단어 시드로 시작합니다(원하시면 다른 케냐 지역 언어로도 시드 교체 가능).

## 실행 방법

```bash
cd "C:\cursor app\kenya-vocab"
npm install
npm run dev
```

## 주요 기능

- **단어장**
  - 단어 **추가/수정/삭제**
  - **검색(sw/ko/en)** 및 **태그 필터**
- **학습(플래시카드)**
  - “정답 보기” 후 **몰라요/애매해요/알아요**로 평가
  - 간단한 **간격 반복(SRS)** 로직으로 다음 복습 시점을 자동 조정
- **통계**
  - 전체 단어, 복습 대상, 오늘 리뷰/정확도 요약
- **설정/백업**
  - “기한 도래(due)만 보기”, “영어(en) 표시” 토글
  - **내보내기/가져오기(JSON)**, **샘플 단어로 리셋**

## 데이터 저장

- 앱 데이터는 브라우저의 **localStorage**에 저장됩니다.
- 장치/브라우저를 옮기기 전에 **설정 → 내보내기(JSON)**로 백업하세요.
